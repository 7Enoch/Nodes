###### 概率证明

```java
 int count = 0;
        int testTimes = 100000;
        for (int i = 0; i < testTimes; i++) {
            if (0.3 > Math.random()) {
                count++;
            }

        }
        System.out.print("概率为:"+( ( double)count/(double)testTimes));
```
